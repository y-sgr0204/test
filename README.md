# マークダウン記法
### 段落  
1段落目  
（空行を挟む）  
2段落目

### 改行  
行末に半角スペースを2つ

### 強調  
強調したい文章をアスタリスク（*）  
あるいは、アンダーバー（_）で挟む  
強い強調は2つで挟む

### コード表示  
バッククォーテーション（'）で挟む  
複数行の場合は、行頭に半角スペースを4個置く

### リスト  
行頭にアスタリスク（*）
あるいは、番号ピリオド（1.）を置く

### 見出し
行頭の#の個数でH1〜H6を示す

### 引用
メールの引用の感じ。  
> "引用引用引用引用引用引用引用引用引用引用引用"

他に2種類の記法あり
